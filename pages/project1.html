Rekoyo

Spread the art of helping

Rekoyo is a mobile application which connects the multigeneration people. User who seek guidance can get by posting status, chatting with other users and by categorized their connections using filters. Users can also refer their connections to another user.



Target Audience

Teenagers to Senior Citizens.



Main features

Categories for connections, User can add their connections into the categories so that they can search easily when they have a need of help. There are 6 default categories like Developer, Designer, Banking, Business, Medical and Others.

Post Status, User can also post status on their profile regarding anything as if they need a help on any topic or they just want to give an advice and post will shown to other user.

Chat, User have a option to chat with each others for the interaction and sharing the experience.

Refer a friend, User can refer the other user profiles to their friends if they need someone specialization for help.

UI Kit



We used blue as a primary and orange as a secondary color for Rekoyo application. We used the Montserrat typeface because of it's visibility and its X-height. We make the icons with round edges so it is easily recognized by the every user. Logo represent the main goal of application like Re represent the Retired, Ko represent the Connestion and Yo represent the Young.
