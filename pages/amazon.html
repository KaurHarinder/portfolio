Amazon focuses on online advertising, cloud computing, e-commerce and artificial intelligence. This is most usable website by the multiple users, so I redesign the some main screens of website as well as application of amazon to make it more convenient for the users. What changes I made in UX

In UX, In mobile version, I change the header and footer UX design. I move the hamburger menu from footer to header because It is experienced great by user in the header. I also give the logo in mobile version in header right side. I move the location from top of body to footer as icon. Because it is invisible on the screen and it can be properly used in the footer as it is properly visible.

In the desktop version, I changed the header and footer only, in the header I move the location from left to right, because it is not looks nice with the logo, it should be a in the right side with other icons so it can be properly used and then it makes layout nicely of the header bar.

I move the links from left side to right on the navigation bar, because there is alsoa hamburger menu and it is experienced and looks good on right side of screen. All other links are also make the header layout nice below the other icons rather some links or icon on the right side some are on the left side. In the footer, I move the back to top button below the descriptions, because its position above it looks like hoch poch.
